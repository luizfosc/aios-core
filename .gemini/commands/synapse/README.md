# Synapse Squad

CRUD command router for managing SYNAPSE domains, rules, and star-commands. Provides structured management of Claude Code's custom command system.

## Structure

- **Manager:** 1 (command router)
- **Tasks:** 6
- **Templates:** 2
- **Utilities:** Custom rule management

## Key Commands

- `*create-domain` — Create new SYNAPSE domain
- `*add-rule` — Add rule to existing domain
- `*edit-rule` — Modify existing rule
- `*toggle-domain` — Enable/disable domain
- `*create-command` — Create new star-command
- `*suggest-domain` — Get domain suggestions

## Key Capabilities

- Domain creation and management
- Rule authoring and editing
- Command scaffolding
- Domain organization and toggling

## Activation

`/synapse:README` or load from `squads/synapse/`

## Notes

This squad automates the management of Claude Code's custom rules and commands, ensuring consistent structure and reducing manual YAML/MD editing.
