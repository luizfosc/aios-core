**Princípio Central:** Cada squad mantém sua independência. O orchestrator apenas ROTEIA e COORDENA.

## Quando Usar Este Squad

### Use HIGH-TICKET MASTERY quando:
- Você quer um **lançamento end-to-end** completo
- Você quer que alguém **orquestre** os especialistas por você
- Você não quer pensar em qual squad chamar
- Você precisa de **múltiplos módulos** coordenados

### Use SQUADS INDIVIDUAIS quando:
- Você sabe **exatamente** qual especialista precisa
- Você quer uma resposta **focada e rápida**
- Você não precisa do pipeline completo
- Você já tem outras etapas prontas

**Exemplo:**
- Só precisa de copy? → Chame `/copywriting-squad:README`
- Só precisa de Big Idea? → Chame `/leandro-ladeira:leandro-ladeira`
- Precisa do pipeline completo? → Chame `/high-ticket-mastery:high-ticket-chief`
