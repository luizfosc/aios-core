# DNA Mental - 8-Layer Architecture

Framework proprietario de mapeamento cognitivo completo. Cada mente possui 8 camadas organizadas em 3 niveis de profundidade.

---

## As 8 Camadas

### Nivel 1: Observable (Camadas 1-4)

| Camada | Nome | O que Captura | Fase do Pipeline |
|--------|------|---------------|------------------|
| **Layer 1** | Behavioral Patterns | Habitos, rotinas, como age no dia a dia | extract-voice-dna (Phase 3: Writing Style) |
| **Layer 2** | Communication Style | Tom, vocabulario, storytelling, emoji usage | extract-voice-dna (Phases 1-5) |
| **Layer 3** | Routines & Habits | Processos repetitivos, rituais de trabalho | extract-thinking-dna (Phase 1: Recognition) |
| **Layer 4** | Recognition Patterns | O que nota primeiro, blind spots, triggers | extract-thinking-dna (Phase 1: Recognition) |

### Nivel 2: Cognitive (Camada 5)

| Camada | Nome | O que Captura | Fase do Pipeline |
|--------|------|---------------|------------------|
| **Layer 5** | Mental Models | Frameworks, heuristicas, decision trees | extract-thinking-dna (Phases 2-4) |

### Nivel 3: Deep Identity (Camadas 6-8)

| Camada | Nome | O que Captura | Fase do Pipeline |
|--------|------|---------------|------------------|
| **Layer 6** | Values Hierarchy | O que defende, o que rejeita, prioridades | extract-voice-dna (Phase 6: Immune System) |
| **Layer 7** | Core Obsessions | Temas recorrentes, "cavalos de batalha" | extract-thinking-dna (Phase 5: Anti-Patterns) |
| **Layer 8** | Productive Paradoxes | Contradicoes que DEFINEM a pessoa | synthesize-mind (Cross-Reference Analysis) |

---

## Checklist de Completude

Use esta checklist durante `synthesize-mind` para verificar se o DNA cobre todas as 8 camadas:

```
[ ] Layer 1 - Behavioral Patterns     → voice_dna.writing_style + tone.by_context
[ ] Layer 2 - Communication Style      → voice_dna.vocabulary + storytelling + tone
[ ] Layer 3 - Routines & Habits        → thinking_dna.recognition_patterns
[ ] Layer 4 - Recognition Patterns     → thinking_dna.recognition_patterns.instant_detection
[ ] Layer 5 - Mental Models            → thinking_dna.frameworks + heuristics
[ ] Layer 6 - Values Hierarchy         → voice_dna.immune_system + anti_patterns
[ ] Layer 7 - Core Obsessions          → thinking_dna.anti_patterns + attention_triggers
[ ] Layer 8 - Productive Paradoxes     → synthesis.voice_thinking_tensions
```

**Regra:** Clone com menos de 6/8 camadas cobertas = INCOMPLETE. Requer re-extraction direcionada.

---

## Relacao entre Camadas e Fidelidade

| Camadas Cobertas | Fidelidade Esperada | Nivel |
|------------------|--------------------:|-------|
| 1-4 (Observable) | 60-75% | Basic |
| 1-5 (+ Cognitive) | 75-85% | Intermediate |
| 1-7 (+ Deep Identity) | 85-95% | Premium |
| 1-8 (Completo) | 93-97% | Elite |

**Insight:** Layers 6-8 sao o que separa um clone generico de um clone que PENSA como a pessoa. Sem elas, o clone imita superficie mas nao replica identidade.

---

## Como Usar

1. **Durante synthesize-mind:** Rodar checklist acima para cada camada
2. **Quando clone esta fraco:** Verificar quais camadas estao vazias
3. **Para priorizar re-extraction:** Focar nas camadas com menor cobertura
4. **Para comparar clones:** Usar numero de camadas cobertas como metrica
