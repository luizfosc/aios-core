# Offer Stack Template

**Pattern:** CP-PP-005
**Source:** Dan Kennedy (Magnetic Marketing) + Stefan Georgi (RMBC)

---

## USP (Unique Selling Proposition)

**Pergunta Kennedy:**
"Por que EU, seu prospect PERFEITO, deveria escolher fazer negocio com VOCE
sobre TODA outra opcao disponivel no mercado, incluindo nao fazer nada?"

```
Minha USP:
_____________________________________________
_____________________________________________

Teste:
- [ ] E especifica (nao generica)?
- [ ] E significativa pro cliente?
- [ ] E defensavel (posso provar)?
- [ ] Diferencia claramente dos concorrentes?
```

## Offer Stack Structure

### Core Offer

```
Nome: [Nome proprietario do produto/servico]
Descricao: [O que e em 1 frase]
Resultado principal: [Transformacao #1]
Valor percebido: R$ ____

Conteudo/Entrega:
- [ ] Item 1: _______________
- [ ] Item 2: _______________
- [ ] Item 3: _______________
```

### Bonus #1 (Acelera resultado)

```
Nome: [Nome do bonus]
Descricao: [1 frase - como acelera o resultado principal]
Valor: R$ ____
Tipo: [ ] Template [ ] Ferramenta [ ] Treinamento [ ] Acesso
```

### Bonus #2 (Remove objecao)

```
Nome: [Nome do bonus]
Descricao: [1 frase - qual objecao remove]
Valor: R$ ____
Tipo: [ ] Template [ ] Ferramenta [ ] Treinamento [ ] Acesso
```

### Bonus #3 (Bonus surpresa / overdelivery)

```
Nome: [Nome do bonus]

Valor: R$ ____
Tipo: [ ] Template [ ] Ferramenta [ ] Treinamento [ ] Acesso
```

```

Motivo REAL (nao artificial):
"________________________________________"

Deadline: ____/____/____
Consequencia de esperar:

```

---

## CTA (Call to Action)

```
Texto do botao: "[Verbo] + [Resultado]"
Ex: "Quero lotar minha agenda agora"

Sub-CTA (abaixo do botao):
"[Reforco de garantia ou urgencia]"
Ex: "Garantia de 7 dias. Sem risco."
```

## 10 Rules Audit (Kennedy)

- [ ] 1. Tem oferta clara? (nao so informacao)
- [ ] 2. Tem razao para responder AGORA?
- [ ] 3. Instrucoes claras de como comprar?
- [ ] 4. Tracking e mensuracao definidos?
- [ ] 5. Brand building sem custo extra?
- [ ] 6. Follow-up planejado (pos-compra)?
- [ ] 7. Copy forte (nao institucional)?
- [ ] 8. Parece direct response (nao corporativo)?
- [ ] 9. Resultado mensuravel? (ROI trackeavel)
- [ ] 10. Disciplina de direct marketing mantida?
