# Carousel Checklist — Instagram

Checklist especifico para carrosseis no Instagram.
Usar JUNTO com o `content-quality-checklist.md` (master checklist deve passar primeiro).

---

## Structure (BLOCKING — todos devem passar)

- [ ] Total de 7-10 slides (range ideal para engajamento)
- [ ] Slide de capa tem hook de pattern-interrupt (max 8 palavras)
- [ ] Slide de capa tem indicador de swipe (seta → e/ou counter 1/N) — `data/visual-rules.md`
- [ ] Slides seguem fluxo narrativo:
  1. **Capa**: Hook (dor ou afirmacao contraria)
  2. **Slides 2-3**: Dor / Erro comum (identificacao)
  3. **Slides 4-5**: Consequencia / Aprofundamento (tensao)
  4. **Slides 6-8**: Direcao / Framework / Solucao (valor)
  5. **Slide final**: CTA explicito
- [ ] Cada slide tem UM ponto claro (nao mistura ideias)
- [ ] Slide final tem CTA explicito e especifico:
  - [ ] "Manda DM com [palavra-chave]" OU
  - [ ] "Salva pra reler quando travar" OU
  - [ ] "Manda pra quem precisa ouvir isso" OU
  - [ ] Link na bio para Next Step

**Resultado Structure:** [ ] PASSA / [ ] BLOQUEADO

## Copy Quality (BLOCKING — todos devem passar)

- [ ] Headlines sao bold e diretas (max 8 palavras por headline)
- [ ] Body text: max 3 frases por slide (nenhuma frase com mais de 20 palavras)
- [ ] Usa pelo menos 1 frase-assinatura do Tiago:
  - "Voce nao precisa aprender mais. Precisa decidir melhor."
  - "Clareza vem antes de disciplina."
  - "O problema nao e falta de acao, e acao sem direcao."
  - "Se tudo parece urgente, nada e importante."
  - "Obesidade mental trava mais que preguica."
- [ ] Zero filler words (remover: "basicamente", "na verdade", "tipo assim", "enfim")
- [ ] Pelo menos 1 slide com angulo contrario ou inesperado:
  - Ex: "Disciplina nao e o problema" (quando todos dizem que e)
  - Ex: "Parar de fazer e mais importante que comecar a fazer"

**Resultado Copy:** [ ] PASSA / [ ] BLOQUEADO

## Visual Design (min 80% para passar)

**Referência**: `data/visual-rules.md` (7 Princípios de Alta Performance)

- [ ] Formato: 1080x1350px (4:5 ratio)
- [ ] Design system consistente em todos os slides:
  - [ ] Fonte headlines: Inter ExtraBold
  - [ ] Fonte body: Courier Prime
  - [ ] Cores: paleta validada (navy/branco alternado + accent gold #C9A84C)
- [ ] Ritmo visual alternado:
  - Slides impares: fundo branco, texto navy
  - Slides pares: fundo navy/escuro, texto branco
- [ ] Legivel no mobile (fonte headline min 48px, body min 28px)
- [ ] Indicadores de slide visiveis (numero ex: 1/7 na esquerda, seta na direita)
- [ ] Espacamento consistente (margins, padding uniformes)
- [ ] Sem poluicao visual (max 2 elementos por slide: headline + body)
- [ ] Contraste adequado (texto legivel sobre qualquer fundo, WCAG AA >= 4.5:1)
- [ ] Cor dominante unica por slide (navy OU branco, nunca ambos disputando)
- [ ] Se capa com foto: textura/grain no fundo (+79% likes), expressao contemplativa

## Engagement Optimization (min 80% para passar)

- [ ] Slide 1 tem swipe trigger (curiosity gap — faz querer ver o proximo)
- [ ] Conteudo e save-worthy (pratico, referenciavel, vale reler)
- [ ] Conteudo e share-worthy (pelo menos 1 slide gera "preciso mandar isso pra alguem")
- [ ] Tem prompt de comentario em algum slide:
  - Ex: "Qual desses erros voce mais comete?"
  - Ex: "Comenta 'clareza' se isso fez sentido"
- [ ] Ultimo slide cria sensacao de completude + acao

**Resultado Engagement:** ___/5 itens (min 4/5 = 80%)

## Caption (NON-BLOCKING mas recomendado)

- [ ] Hook da caption complementa (nao repete) o hook do carousel
- [ ] Primeiras 125 caracteres sao compelling (visivel antes do "mais")
- [ ] Inclui CTA de save: "Salva esse post pra reler quando travar"
- [ ] Inclui CTA de share: "Manda pra quem precisa de direcao"
- [ ] Hashtags: 5-10, mix de:
  - Alto volume (2-3): #empreendedorismo, #produtividade, #mentalidade
  - Medio (3-4): #clarezamental, #foco, #direcao
  - Nicho (2-3): #obesidademental, #mapadedirecao
- [ ] Sem hashtags irrelevantes ou spam
- [ ] Emoji: max 3, apenas se adicionam significado (nao decorativos)

## Quick Reference — Erros Comuns

1. **Slide de capa generico** — "5 dicas para ser mais produtivo" NAO funciona. Usar angulo contrario.
2. **Slides com texto demais** — Se precisa de mais de 3 frases, dividir em 2 slides.
3. **CTA vago** — "Me siga para mais" e fraco. Ser especifico: "Salva e releia quando travar."
4. **Sem tensao** — Pular direto para a solucao sem estabelecer a dor. A dor vende a solucao.
5. **Tom motivacional** — "Voce consegue!" nao e o tom. O tom e: "Para de se enganar. Faz o seguinte..."

## Auto-Correcao — Se Falhar, Faca Isso

| Falha | Correcao |
|-------|----------|
| Structure — Capa sem pattern-interrupt | Reescreva com angulo contrario em max 8 palavras. Ex: "Disciplina nao resolve nada" |
| Structure — Fluxo narrativo quebrado | Reorganize: Capa(hook) → Dor(2-3) → Tensao(4-5) → Valor(6-8) → CTA(final) |
| Structure — Mais de 1 ponto por slide | Divida o slide em 2. Cada slide = 1 ideia unica |
| Copy — Headlines longas | Corte para max 8 palavras. Remova artigos e preposicoes desnecessarios |
| Copy — Body com mais de 3 frases | Mova excesso para o proximo slide ou delete |
| Copy — Sem frase-assinatura | Insira 1 frase do Tiago no slide de valor (6-8) |
| Copy — Filler words presentes | Busque e remova: "basicamente", "na verdade", "tipo assim", "enfim" |
| Visual — Fonte incorreta | Headlines: Inter ExtraBold. Body: Courier Prime. Sem excecoes |
| Visual — Alternancia B&W errada | Impares=branco, Pares=navy/escuro. Verifique cada slide |
| Visual — Ilegivel no mobile | Headline min 48px, body min 28px. Reduza texto se necessario |
| Visual — Cor dominante disputada | Cada slide = 1 cor dominante (navy OU branco). Ver `data/visual-rules.md` |
| Visual — Capa flat sem textura | Adicione grain/noise. No Nano Banana: "film grain" no Bloco 5 |
| Engagement — Sem swipe trigger | Adicione curiosity gap no slide 1: promessa que so se resolve nos proximos slides |
| Engagement — Sem prompt de comentario | Adicione pergunta em 1 slide: "Qual desses erros voce mais comete?" |
| Caption — Hook repete carousel | Reescreva caption com angulo complementar, nao repetido |
