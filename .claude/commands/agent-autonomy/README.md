# Agent Autonomy Squad

> **Audite, crie, diagnostique e otimize agentes autonomos.**

Baseado em 14 elite minds (Lilian Weng, Erik Schluntz, Harrison Chase, etc.)

## Subcomandos

| Comando | Descricao |
|---------|-----------|
| `/agent-autonomy:chief` | Ativa orchestrador (triage automatico) |
| `/agent-autonomy:audit` | Auditar agente existente |
| `/agent-autonomy:create` | Criar agente autonomo |
| `/agent-autonomy:diagnose` | Diagnosticar falhas de autonomia |
| `/agent-autonomy:optimize` | Otimizar agente existente |
| `/agent-autonomy:teach` | Ensinar reasoning patterns |
| `/agent-autonomy:find` | Pesquisar libs/repos/benchmarks |
| `/agent-autonomy:tools` | Sugerir/construir tools |

## Fonte

`squads/agent-autonomy/` — Config: `squads/agent-autonomy/config/config.yaml`
