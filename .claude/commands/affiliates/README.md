# Affiliates Squad — Multi-Platform Affiliate Business System

## Overview

Squad completo para pesquisar, gerir e executar negócios de afiliados em múltiplas plataformas: Amazon, Shopee, Mercado Livre, Hotmart, Kiwify, ClickBank, mesas proprietárias (FTMO, Apex) e plataformas internacionais.

17 agents especializados (13 clones de elite minds + 4 funcionais) cobrindo o ciclo completo: **pesquisar → decidir → executar → medir → otimizar → escalar/pausar**.

## Purpose

Este squad transforma o negócio de afiliados de "tentativa e erro" para uma operação sistemática baseada em frameworks documentados de elite minds reais — pessoas com skin in the game e resultados comprovados.

## When to Use This Pack

Use `affiliates` quando você quer:

- Iniciar um negócio de afiliados do zero (diagnóstico + estratégia)
- Criar sites de review/comparativo para Amazon, Shopee, MeLi
- Montar funnels para captar leads de prop trading
- Criar conteúdo para TikTok/Reels/YouTube que converte em vendas
- Estruturar email marketing para nurturing de afiliados
- Analisar métricas e decidir onde escalar ou pausar
- Operar como afiliado Hotmart/Kiwify no Brasil
- Montar grupos de ofertas no WhatsApp/Telegram

## Usage Examples

### Exemplo 1: Comecar do Zero

```
@affiliates:affiliates-chief
> "Quero comecar como afiliado, nao sei por onde"

→ Chief roteia para affiliate-strategist
→ Diagnostico: plataforma, nicho, modelo
→ Handoff para agent especializado (ex: seo-affiliate ou affiliate-br)
```

### Exemplo 2: Otimizar Negocio Existente

```
@affiliates:performance-analyst
> "Meu site Amazon ganha R$3K/mes mas estagnou"

→ STDC audit (See-Think-Do-Care)
→ Identifica gargalo (ex: trafego estagnado)
→ Handoff para growth-optimizer (ICE scoring para proximos passos)
```

### Exemplo 3: Criar Conteudo Social

```
@affiliates:social-hooks
> "Preciso de hooks para review de produto no TikTok"

→ 20 variações de hook usando Hook Point framework
→ HSA (Hook-Story-Authenticity) para cada variacao
→ Plano de teste HTP (Hypothesis-Test-Pivot)
```
