## 💬 Sistema de Debate Entre Agentes

### O Diferencial Deste Squad

**TODOS os agentes DISCUTEM e DEBATEM** para chegar à melhor solução.

### Como Funciona

```
PROPOSTA → DEBATE → VOTAÇÃO → CONSENSO → AÇÃO

Exemplo:
@hook-master propõe: "Hook 1: Você está fazendo errado"
@retention-architect: "Hook 1 retém 82% @3s (dados)"
@viral: "Hook 1 pode alienar. Hook 2 mais inclusivo?"
@metrics-guru: "Dados mostram controversy = +40% engagement"

VOTAÇÃO PONDERADA:
@viral (peso 3x): Hook 1
@retention-architect (peso 3x): Hook 1
@metrics-guru (peso 3x): Hook 1
RESULTADO: Hook 1 vence

CONSENSO: ✅ Hook 1 aprovado
```

### Pesos de Votação

- **Peso 3x:** @viral, @retention-architect, @metrics-guru
- **Peso 2x:** @hook-master, @visual-impact, @thumbnail-king, @algorithm-hacker, @engagement-engineer, @remotion-architect, @growth-hacker, @oalanicolas, @pedro-valerio
- **Peso 1.5x:** @motion-master, @animation-pro, @sound-designer
- **Peso 1x:** Demais agentes

### Regras de Consenso

- **Unanimidade preferencial** para decisões críticas
- **Maioria ponderada (>60%)** para decisões operacionais
- **Veto power:** @viral pode vetar com justificativa estrategica; @pedro-valerio pode vetar por falha de processo
- **Data trumps opinion:** Dados de @metrics-guru sempre pesam mais
- **DNA trumps generic:** Insights de @oalanicolas pesam mais que conteudo generico

### Pontos de Debate Obrigatórios

1. Seleção de trend

## 🎨 Design System Academia Lendária v4.1

### Minimalismo Lendário

**Colors:**

```
Background: #000000 (~70%)
Foreground: #FFFFFF (~22%)
Primary Gold: #C9B298 (MAX 8%) ⚠️
Muted: #A8A8A8
```

### 8% Gold Rule (SACRED)

**NUNCA exceder 8% de ouro na tela.**
Usar APENAS para:

- CTAs principais
- Key highlights
- Important numbers
- Accent borders

### Typography

```
UI/Títulos: Inter SemiBold 600
Corpo: Source Serif 4 Regular 400
```
