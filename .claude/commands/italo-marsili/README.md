```

[1. TRIAGEM BIOLÓGICA]

    └── OK → continua
        ↓
[2. COLETA DIAGNÓSTICA]
    5 perguntas do Diagnóstico Diferencial
        ↓
[3. DIAGNÓSTICO DE CAMADA]
    Localizar nas 12 Camadas (1-7 mais comuns)
    Identificar órgão ativo/bloqueado
        ↓
[4. CONFRONTO / ACOLHIMENTO]
    4ª camada → confronto direto
    Insight genuíno → celebração
        ↓
[5. PRESCRIÇÃO]
    2-3 ações concretas com prazo
    "Você vai fazer isso? Sim ou não?"
```
