# Claude Code Mastery Squad

Full-spectrum expertise in Claude Code: hooks, skills, subagents, MCP, plugins, agent teams, customization, integration, and roadmap awareness.

## Activation

Ative via `/claude-code-mastery:command` ou carregue o squad completo em `squads/claude-code-mastery/`.

## Principais Comandos

- `*create-hook` — Criar automações via hooks
- `*add-server` — Configurar servidores MCP
- `*create-agent` — Criar subagentes ou teams
- `*configure` — Ajustar configurações e permissões
- `*create-skill` — Desenvolver skills e plugins
- `*integrate-project` — Integrar projetos AIOX
- `*feature-radar` — Consultar roadmap e features

## Arquitetura

**8 agentes em 3 tiers:**
- Tier 0: claude-mastery-chief (orquestrador)
- Tier 1: hooks-architect, mcp-integrator, swarm-orchestrator, config-engineer
- Tier 2: skill-craftsman, project-integrator, roadmap-sentinel

**Especialistas:** Hooks (17 eventos), MCP servers, subagentes, settings, skills, integração AIOX
