# WhatsApp Prospector

Squad especializado em prospectar leads qualificados a partir de exports de grupos de WhatsApp, cruzando com soluções do cliente (5 pilares), ICPs, red flags e gerando mensagens personalizadas de outreach para Google Sheets.

**Version:** 3.0.0 | **Entry Agent:** Atlas (prospector-chief) | **Model Tier:** haiku/sonnet/opus

---

## Pipeline

1. Parse chat export → extrair participantes e mensagens
2. Resolve phone numbers → associar nomes a números
3. Analyze prospects → scoring com ICP/red flags
4. Write outreach → mensagens personalizadas
5. Populate sheet → output para Google Sheets
