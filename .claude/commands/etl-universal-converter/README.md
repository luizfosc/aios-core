# ETL Universal Converter

Content ETL pipeline que converte qualquer formato de arquivo em markdown tratado e normalizado com frontmatter padronizado.

## Capabilities

- **Classificacao automatica** por extensao + MIME type
- **13 tasks** especializadas (convert, classify, validate, enrich, curate)
- **4 workflows** (single, batch, purpose-driven, quality-review)
- **Quality scoring** com veto conditions e human review triggers

## Supported Formats

Video/Audio, PDF/EPUB/MOBI, CSV/Excel, PowerPoint, Imagem (OCR), JSON/YAML/XML, TXT/MD

## Key Commands

- `*convert` — Converter arquivo unico
- `*batch` — Converter lote de arquivos
- `*classify` — Classificar tipo de arquivo
- `*validate` — Validar qualidade da extracao
