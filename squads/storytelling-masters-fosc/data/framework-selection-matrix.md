## Multi-Expert Consultation Protocol

**When to consult multiple experts:**

### Scenario 1: High-Stakes Keynote (e.g., Product Launch, TED, Conference Closing)
**Consult:**
1. **Duarte** (structure) → Sparkline, S.T.A.R. moments
2. **Gallo** (delivery) → 9 TED Secrets, jaw-dropping moment
3. **Heath** (stickiness) → SUCCESs audit
4. **McKee** (if story-heavy) → Beat validation, value changes

**Integration:** Duarte builds the skeleton → Gallo adds the muscles (delivery) → Heath adds the nerves (memorable touches) → McKee ensures every scene turns.

---

### Scenario 2: Sales Presentation (Demo + Pitch)
**Consult:**
1. **Miller** (message clarity) → SB7 framework
2. **Cialdini** (persuasion tactics) → Authority, Social Proof, Scarcity
3. **Duarte** (structure) → Sparkline oscillation
4. **Heinrichs** (argumentation) → Ethos establishment, concession strategy

**Integration:** Miller clarifies the message → Duarte structures the talk → Cialdini provides persuasion levers → Heinrichs handles objections.

---

### Scenario 3: Personal Story (Vulnerability + Transformation)
**Consult:**
1. **Dicks** (core structure) → 5-Second Moment, Storyworthy components
2. **Campbell** (archetypal depth) → Monomyth stages
3. **McKee** (scene validation) → The Gap, turning points
4. **Carnegie** (delivery fundamentals) → Authenticity, audience connection

**Integration:** Dicks finds the emotional core → Campbell maps the journey → McKee ensures structure integrity → Carnegie guides delivery.

## Framework Layering Guidelines

### Layer 1: FOUNDATION (always apply first)
Choose ONE foundational framework based on primary goal:
- **Story:** McKee Story Principles
- **Presentation:** Duarte Sparkline
- **Persuasion:** Heinrichs Ethos/Pathos/Logos
- **Personal Story:** Dicks 5-Second Moment
- **Brand Message:** Miller StoryBrand

### Layer 2: ENHANCEMENT (add for depth)
- **Archetypal resonance:** Campbell Monomyth
- **Stickiness:** Heath SUCCESs
- **Persuasion tactics:** Cialdini 6 Principles
- **Delivery mastery:** Gallo TED Secrets

### Layer 3: POLISH (final touches)
- **Memorable moments:** Duarte S.T.A.R. / Heath EPIC
- **Rhetorical flourish:** Heinrichs Figures of Speech
- **Authenticity:** Carnegie Fundamentals / Dicks Vulnerability

**Anti-Pattern:** Applying all frameworks equally = analysis paralysis. Start with ONE foundation, layer strategically.
