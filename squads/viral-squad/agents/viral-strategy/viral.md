## Viral Frameworks

### 1. Viral Video Formula

```
Hook (0-3s) → Problem Agitation (3-8s) → Solution Tease (8-15s) →
Value Delivery (15-45s) → Payoff (45-55s) → CTA (55-60s)
```

### 2. Share Triggers Checklist

Vídeo tem pelo menos 3 destes?

- [ ] Shocking/Surprising element
- [ ] Makes viewer look smart/cool
- [ ] Practical value (save for later)
- [ ] Strong emotion (awe, anger, humor)
- [ ] Identity expression (this is me)
- [ ] Controversy/Debate starter
- [ ] Before/After transformation
- [ ] Secret/Insider knowledge

### 3. Viral Coefficient Formula

```
Viral Coefficient = (Shares × Avg Followers per Sharer) / Total Reach

Target: >1.5 for explosive growth
```

### 4. Platform-Specific Optimization

**Instagram Algorithm Priorities (2025):**

1. **Saves** (strongest signal) - marca como "valuable"
2. **Shares** (viral signal) - quer que mais pessoas vejam
3. **Comments** (engagement) - conversa ativa
4. **Completion Rate** (quality) - prende até o fim
5. **Profile Visits** (interest) - quer conhecer mais
6. **Follows** (conversion) - fan verdadeiro
