# User Guide - Copywriting Masters Squad

## Introducao

Bem-vindo a Copywriting Masters Squad! Este guia vai te ensinar a usar os 15 clones de copywriters lendarios para criar copy de alta conversao.

---

## Conceito Central

### A Hierarquia de 3 Tiers

Imagine que voce esta construindo uma casa:

1. **Tier 1 - Estrategistas (Arquitetos)**
   - Definem a planta antes de construir
   - Usar ANTES de escrever qualquer copy

2. **Tier 2 - Executores (Pedreiros)**
   - Constroem baseado na planta
   - Usar PARA escrever o copy

3. **Tier 3 - Otimizadores (Inspetores)**
   - Verificam a qualidade final
   - Usar DEPOIS de escrever

### O Erro Mais Comum

Pular direto para Tier 2 (escrever) sem passar pelo Tier 1 (estrategia).

**Resultado:** Copy bonita que nao converte porque nao foi pensada estrategicamente.

## Fluxo de Trabalho Padrao

### Passo 1: Diagnostico de Awareness (OBRIGATORIO)

Sempre comece com Eugene Schwartz:

```
@eugene-schwartz
```

Responda as perguntas dele para descobrir em qual dos 5 niveis seu publico esta:

1. **Unaware** - Nao sabem que tem problema
2. **Problem Aware** - Sabem do problema
3. **Solution Aware** - Conhecem solucoes
4. **Product Aware** - Conhecem voce
5. **Most Aware** - Prontos para comprar

Isso define TODO o resto do seu copy.

Depois de escrever, adicione gatilhos psicologicos:

```

```

Escolha 5-10 triggers relevantes e injete no copy.

Antes de publicar, faca audit com Hopkins:

```

```

Ele vai passar um checklist rigoroso no seu copy e identificar pontos de melhoria.

## Dicas Avancadas

### 1. Combine Clones para Maior Impacto

Algumas combinacoes funcionam muito bem juntas:

- **Halbert + Makepeace:** Historia + Emocao = Sales letter poderosa
- **Todd Brown + Hormozi:** Mecanismo + Oferta = Diferenciacao completa
- **Chaperon + Settle:** Automacao + Broadcast = Sistema de email completo

### 2. Evite Estas Combinacoes

- **Halbert + Carlton na mesma peca:** Estilos conflitantes
- **Ogilvy + Makepeace:** Sofisticado vs Visceral
- **Todos os 31 triggers de uma vez:** Fica artificial

### 3. Adapte a Extensao ao Awareness

| Awareness | Extensao |
|-----------|----------|
| Unaware | Copy LONGO (educar muito) |
| Problem Aware | Copy MEDIO-LONGO |
| Solution Aware | Copy MEDIO |
| Product Aware | Copy CURTO |
| Most Aware | Copy MUITO CURTO (oferta direta) |

### 4. Teste Sempre (Hopkins)

Nunca assuma que sabe o que funciona. Teste:

1. Headlines (maior impacto)
2. Oferta (segundo maior impacto)
3. Lead/Abertura (terceiro)
4. CTA (quarto)
