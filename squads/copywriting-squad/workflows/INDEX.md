## ★ WORKFLOW OBRIGATÓRIO

**Fases:**
1. Fundação Estratégica (Schwartz + Kennedy + Todd Brown + Hormozi)

**Quando usar:** SEMPRE que criar uma sales page. Sem exceções.

## Fluxo de Uso Recomendado

**Para lançamento completo:**
```
1. Oferta Irresistível (criar oferta)
2. Sales Page Killer ★ (sales page de classe mundial)
3. VSL Matador (criar vídeo de venda)
4. Máquina de Leads (capturar leads)
5. Webinar que Vende (converter leads)
6. Email Marketing Completo (monetizar lista)
```

**Para high-ticket:**
```
1. High-Ticket Sales (workflow completo de 3 fases)
   └── Inclui: oferta + webinar + sales page + call + emails
```

**Para tráfego pago rápido:**
```
1. Oferta Irresistível (definir oferta)
2. Ads que Escalam (criativos + LP + testes)
3. Máquina de Leads (capturar leads)
```

**Para monetizar lista existente:**
```
1. Reativação de Lista (limpar e engajar)
2. Email Marketing Completo (setup sistema)
```

**Para construir audiência orgânica:**
```
1. Conteúdo Orgânico Diário (sistema semanal)
2. Email Marketing Completo (capturar e monetizar)
```

**Para otimizar o que já existe:**
```
1. Otimização de Funil (diagnosticar + corrigir + testar)
```

## Estrutura de Cada Workflow (Framework 9 Passos)

```
PASSO 1: INPUT — O que precisa ter antes
PASSO 2: INPUT VALUE — Formato específico dos inputs
PASSO 3: PRE-CHECKLIST — Quality gate de entrada
PASSO 4: PERSONA/EXECUTOR — Qual clone executa
PASSO 5: PROMPT DE EXECUÇÃO — Instruções detalhadas
PASSO 6: TOOLS/SCRIPTS — Ferramentas e dependências
PASSO 7: OUTPUT — O que sai (formato exato)
PASSO 8: POS-CHECKLIST — Quality gate de saída
PASSO 9: OUTPUT FINAL — Resultado completo
```
