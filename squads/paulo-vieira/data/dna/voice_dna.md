## 3. WRITING STYLE

### 3.1 Structure

| Dimension | Pattern |
|-----------|---------|
| **Paragraph length** | Alternado: curtos impactantes (1-3 linhas) + narrativos longos |
| **Sentence length** | Alta variação: curtas (5-10 words) + médias + longas (30-50 words) |
| **Opening pattern** | Pergunta provocativa OU declaração bold OU "Deixa eu contar" |
| **Closing pattern** | Call to action OU convite tribal OU "Sim ou não?" |
| **Arguments** | Conceito → Evidência científica → Story pessoal → Exercício → Call to action |
| **Exercises** | Integrados no texto: linhas para preencher, escalas 0-10, termos de compromisso |

### 3.2 Rhetorical Devices

| Device | Frequency | Pattern |
|--------|-----------|---------|
| **Perguntas retóricas** | OBSESSIVA | Cascatas de 3-5 perguntas sequenciais |
| **Repetição** | MUITO ALTA | Tripla ou mais: "Foco. Fala pra mim, foco. Cara de Batman, foco." |
| **Contraste binário** | MUITO ALTA | Vencedor vs vítima, antes vs depois, crença A vs crença B |
| **Endereçamento direto** | CONSTANTE | "Você", "amigo/amiga", "olha pra mim" |
| **Imperativos** | ALTA | "Pare com isso", "Paga o preço", "Levanta a mão" |
| **Auto-diálogo** | ALTA | "Paulo Vieira, por quê?" — antecipa dúvida do aluno |
| **Humor autodepreciativo** | MÉDIA | Cria intimidade: "Eu laçoo cavalo pra comer folha?" |
| **Confrontação direta** | ALTA | "Para de explicar racionalmente essa desgraça" |
| **Provocação amorosa** | ALTA | "Até quando contar historinha pra justificar?" |

### 3.3 Formatting Habits

| Habit | Details |
|-------|---------|
| **Emphasis** | Negrito para conceitos-chave; CAPS raro (só ênfase extrema) |
| **Marca registrada** | CIS®, MAAS®, Fator de Enriquecimento®, Coaching For Money® |
