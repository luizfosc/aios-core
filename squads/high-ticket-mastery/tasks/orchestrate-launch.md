# Task: Orchestrate Launch

## Metadata
- **Type:** Orchestration
- **Specialist:** High-Ticket Chief
- **Duration:** 6-8h (full pipeline)
- **Activation:** `/high-ticket-mastery:workflows:end-to-end-launch`

## Purpose

Orquestrar lançamento high-ticket completo end-to-end, coordenando 8 especialistas sequencialmente do posicionamento ao fechamento.

## Inputs

- Nicho/produto
- Público-alvo
- Budget disponível (opcional)
- Materiais existentes (se houver)

## Process

1. Ativar workflow: `/high-ticket-mastery:workflows:end-to-end-launch`
2. Executar 8 fases sequencialmente:
   - Fase 1: Posicionamento (Renan Vieira)
   - Fase 2: Big Idea (Leandro Ladeira)
   - Fase 3: Oferta (Alex Hormozi)
   - Fase 4: Copy (Copywriting Squad)
   - Fase 5: VSL (Renner Silva)
   - Fase 6: Tráfego (Tessman/Ladeira)
   - Fase 7: Negociação (Chris Voss)
   - Fase 8: Comunicação (Tathi Deandhela)
3. Validar checkpoint após cada fase
4. Handoff automático para próxima fase

## Outputs

- **Posicionamento:** Estratégia de autoridade percebida
- **Big Idea:** Hipótese central validada
- **Oferta:** Grand Slam Offer completa
- **Copy:** Sales page + emails + ads
- **VSL:** Roteiro completo
- **Tráfego:** Campanha Google/Meta Ads
- **Negociação:** Script de fechamento + objeções
- **Comunicação:** Refinamentos de rapport

## Checkpoint

**Condition:** Todas as 8 fases completas com outputs entregues

**Veto Conditions:**
- Fase skippada (todas são obrigatórias)
- Checkpoint de fase não passou
- Specialist não disponível
