#### ✅ 5 FASES DO PERPÉTUO — Estrutura de Tráfego

**Funil horizontal:** Descoberta → Relacionamento → Conversão → Remarketing → Recuperação

```
FASE 1: DESCOBERTA (anúncios de explicação, captura 75%)
   ↓
FASE 2: RELACIONAMENTO (aquecimento, captura 25%)
   ↓

   ↓
```

- **Iniciante:** Fase 3 + 4 (mínimo viável)

#### ✅ PIRÂMIDE DE PRONTIDÃO — Dinâmica de Vendas

**Estrutura invertida:** Urgência (topo) → Consciência (meio) → Oportunidade (base)

```
         ▲ URGÊNCIA (menor público, maior prontidão)
        / \  Vende: O PRODUTO
       /   \  Busca: "Curso de X"
      /     \  Analogia: Pescando no aquário
     /       \ 
    /         \ CONSCIÊNCIA (público médio)
   /           \  Vende: A RESOLUÇÃO
  /             \  Busca: "Como resolver X"
 /               \  Analogia: Pesque-pague
/                 \
/_________________\ OPORTUNIDADE (maior público, menor prontidão)
                    Vende: A IDEIA
                    Busca: Nada específico
                    Analogia: Pescando no oceano
```

**SEMPRE começar por URGÊNCIA** (menor risco, monetização imediata, dados rápidos)
