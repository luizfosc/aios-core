## 🔑 INSIGHTS PRINCIPAIS

### 1. Sistema Operacional = Perpétuo + Lançamento

Leandro NÃO escolhe entre um ou outro — usa os DOIS:
- **Perpétuo:** Motor sempre ligado (vendas diárias, dados, inteligência)
- **Lançamento:** Turbo boost (pressão, decisão forçada)

### 2. SEMPRE Começar por Urgência

Pirâmide de Prontidão:
- Urgência (topo) → Consciência (meio) → Oportunidade (base)
- Nunca começar pela base (risco alto, custo alto, dados ruins)
- Sempre começar pelo topo (risco baixo, monetização imediata, aprendizado rápido)

### 3. Regra de 4 Dias

Otimizar campanhas a cada 4 dias (NUNCA antes):
- Razão: Precisa de dados para decisão assertiva
- Mexer antes = decisão emocional, não baseada em dados

### 4. Inteligência > Vendas

No VME, o objetivo primário é INTELIGÊNCIA:
- Leads, CPL, LPV, CPA, ROAS, ROI
- Criativos vencedores
- Retenção de conteúdo
- Vendas são consequência, não objetivo

### 5. "Não Dê Ponto Sem Nó"

Todo conteúdo precisa de:
- **M**omento (onde a pessoa está)
- **T**ipo (formato adequado)
- **O**bjetivo (o que você quer alcançar)
