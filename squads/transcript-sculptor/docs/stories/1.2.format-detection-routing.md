## Story

**As a** user,
**I want** to point at a folder with mixed files and have the system automatically detect the type of each file,
**so that** I don't need to manually convert anything before processing.

## Acceptance Criteria

1. Recursive scan of input folder detects: .md, .pdf, .docx, .epub, .txt, .srt, .vtt, .rtf, .odt
2. Each file is classified and routed to the appropriate converter function
3. Unsupported files listed in a skip report (not an error — graceful skip)
4. Processing log generated with status of each file (success/skipped/error)
5. `ingestion-report.yaml` generated per architecture spec

## Dev Notes

### Architecture — Ingestion Report Schema

```yaml
ingestion_report:
  input_folder: "/path/to/input"
  processed_at: "ISO-8601"
  files:
    - original: "filename.ext"
      format: "markdown|pdf-digital|pdf-scanned|docx|epub|srt|vtt|txt|rtf|odt"
      converter: "passthrough|pymupdf|docling+surya|pandoc|subtitle-parser|skipped"
      output: "raw/filename.md"
      status: "success|skipped|error"
      word_count: 12345
  summary:
    total_files: N
    processed: N
    skipped: N
    total_words: N
```

### Converter Routing Table

| Extension | Format | Converter | Script |
|-----------|--------|-----------|--------|
| .md, .txt | text | passthrough | — |
| .pdf | pdf-digital or pdf-scanned | pymupdf or docling+surya | `convert_pdf.py` |
| .docx | docx | pandoc | `convert_docs.py` |
| .epub | epub | pandoc | `convert_docs.py` |
| .rtf, .odt | document | pandoc | `convert_docs.py` |
| .srt, .vtt | subtitle | subtitle-parser | `convert_subtitles.py` |

### Runtime Working Directory

```
{output_folder}/
├── raw/           ← converted .md files go here
└── ingestion-report.yaml
```

### Key Implementation Details

- **Format Detection:** Extension-based with MIME type fallback (mimetypes module)
- **Edge Cases Handled:**
  - Hidden files (starting with .) → skipped with reason
  - Symlinks → skipped with reason
  - Files without extension → skipped with reason
  - Unsupported formats → skipped with reason
  - Empty folders → report with 0 files processed
- **Passthrough Converter:** Uses `shutil.copy2` to preserve metadata
- **Word Counter:** UTF-8 encoding, simple whitespace split
- **Report Format:** YAML with PyYAML library
- **Output Structure:** Creates `raw/` directory automatically
