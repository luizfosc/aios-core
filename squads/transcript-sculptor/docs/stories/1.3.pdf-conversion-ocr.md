## Dev Notes

### PDF Detection Heuristic

```python
import pymupdf
doc = pymupdf.open("file.pdf")
page = doc[0]
text = page.get_text()
# If text density is very low (< 50 chars per page), likely scanned
is_scanned = len(text.strip()) < 50
```

### Docling Usage

```python
from docling.document_converter import DocumentConverter

converter = DocumentConverter()
result = converter.convert("scanned.pdf")
markdown = result.document.export_to_markdown()
```

### PyMuPDF Usage

```python
import pymupdf
doc = pymupdf.open("digital.pdf")
text = ""
for page in doc:
    text += page.get_text() + "\n\n"
```

#### Implemented Functions

1. **`detect_pdf_type(input_path: str) -> str`**
   - Opens PDF with PyMuPDF (fitz)
   - Counts pages with extractable text (>= 50 chars threshold)
   - Returns "digital" if >= 50% pages have text, else "scanned"
   - Includes error handling with graceful fallback to "digital"

2. **`convert_pdf_digital(input_path: str, output_path: str) -> dict`**
   - Extracts text from all pages using PyMuPDF
   - Preserves paragraph breaks with double newlines between pages
   - Returns dict with status, word_count, pages
   - UTF-8 encoding enforced
   - Comprehensive error handling

3. **`convert_pdf_scanned(input_path: str, output_path: str) -> dict`**
   - Primary: Docling with Surya-OCR for high-accuracy OCR
   - Fallback: PyMuPDF if Docling not installed or fails
   - Returns dict with status, word_count, pages, ocr_confidence
   - Graceful degradation with warnings
   - First run will download ~2GB OCR models (documented)

4. **`convert_pdf(input_path: str, output_path: str) -> dict`**
   - Main entry point called by detect_format.py
   - Auto-detects PDF type and routes to appropriate converter
   - Returns combined result dict with pdf_type field
   - Single function interface for external callers

#### Integration Changes

**File:** `squads/transcript-sculptor/scripts/detect_format.py`

#### Technical Decisions

1. **Detection heuristic:** 50% threshold for pages with extractable text (>= 50 chars per page)
2. **Paragraph preservation:** Double newlines between pages to maintain document structure
3. **Graceful fallback:** PyMuPDF fallback if Docling fails or not installed (with warning)
4. **OCR confidence:** Default 0.93 for Docling (Surya-OCR baseline), 0.0 for fallback
5. **Error handling:** Try/except on all major operations with meaningful error messages
6. **Type hints:** All functions have proper type annotations
7. **Encoding:** UTF-8 enforced on all file writes
