## Story

**As a** user,
**I want** the transcription structured with titles, subtitles, proper punctuation, and formatted dialogues,
**so that** the document is navigable and editorial-quality without losing the speaker's DNA.

## Acceptance Criteria

1. H2 titles for each main thematic section (from analysis-map title_suggestion)
2. H3 subtitles for sub-topics within sections
3. Context brackets for non-lecture zones: [Q&A], [Musica], [Pitch de vendas], [Intervalo]
4. Dialogues formatted with **Speaker Name:** before each speech
5. Punctuation adjusted per inferred tone (energetic → !, calm → ., rhetorical → ?)
6. Noise removed (murmurs, contextless loose phrases where confidence >= 0.85)
7. Ambiguous excerpts marked with `[REVISAR]` (confidence < 0.85)
8. Linguistic DNA preserved: vocabulary, expressions, catchphrases, speech rhythm, formality level

```markdown
## Section Title

**Speaker Name:**

Content here...
```

### DNA Preservation Rules (CRITICAL)

1. **NEVER** change the speaker's vocabulary — if they say "galera", keep "galera"
2. **NEVER** formalize informal speech — preserve the register
3. **NEVER** summarize or condense — every substantive sentence stays
4. **paraphrased_sentences MUST be 0** — this is the hard constraint
5. Punctuation can be adjusted (. → ! for energetic tone) but words cannot
6. Filler words ("né", "tipo", "enfim") may be kept if characteristic of speaker
7. Repetitions that serve emphasis ("muito, muito importante") MUST be preserved

### Zone Context Brackets

```markdown
---

## [Q&A] Perguntas e Respostas — Bloco 1

**Participante:** Como funciona na prática?

**Palestrante:** Ótima pergunta! Olha, na prática...

---

## [Intervalo]

_Música de fundo durante o intervalo._

---
```

### Key Implementation Details

**DNA Preservation (Core Constraint):**
- `paraphrased_sentences MUST be 0` — hard constraint enforced in validation
- All catchphrases from analysis-map must be preserved
- Vocabulary immutable (no synonyms, no formalization)
- Repetitions for emphasis preserved ("muito, muito importante")
- Only punctuation can be adjusted, never words

**Integration with Story 2.1:**
- Uses `analysis-map.yaml` produced by transcript-analyst
- Matches chunks by `chunk_id` or word range
- Extracts zones, speakers, noise_candidates, title_suggestions
- Outputs to `chunks/{source}-chunk-{id}-edited.md`

**YAML Frontmatter Structure:**
```yaml
chunk_id: "chunk-001"
source: "raw/palestra-dia1.md"
word_count_before: 7500
word_count_after: 7320
words_removed: 180
removal_log:
  - type: "noise" | "review"
    text: "{excerpt}"
    reason: "{justification}"
    action: "[REVISAR]" (if type: review)
dna_indicators:
  catchphrases_preserved: ["{list}"]
  tone_adjustments: {count}
  paraphrased_sentences: 0  # MUST be 0
```

**Quality Gates:**
- DNA preservation score >= 9.0/10 (blocking)
- Content preservation >= 98% (blocking)
- Structural coherence >= 8.5/10 (blocking)
- Critical rules violation → automatic fail
