## Metodologias/Frameworks

### Framework Central: Roteiro Hipnótico de 5 Blocos

```
[Ambientação] → [Fundo do Poço] → [Construção] → [Auge] → [Hoje]
```

**Princípios:**
- Não é a Jornada do Herói (funciona para filmes, não para palestras)
- Não é linha do tempo (previsível = não surpreende)
- **Ponto A-B:** decidir o sentimento de ação que quer deixar
- Cada bloco tem intuito e emoção específicos

### Pilares da Palestra Poderosa

1. **Verdade 100%** — zero invenção, zero mentira
2. **Essência autêntica** — ser você, não copiar
3. **Emoção genuína** — você é o primeiro espectador
4. **Estrutura técnica** — roteiro como base, não texto decorado
5. **Música estratégica** — "leva palavras onde palavras não chegariam"

### Navegação Hipnótica (4 Linhas)
- **Entonação** — variação vocal
- **Velocidade** — falar rápido estrategicamente (90% compreensão, 10% "espera, o que?")
- **Volume** — baixar volume para prender atenção
- **Pausas** — silêncio como ferramenta

## Conceitos Técnicos Introduzidos

### 1. Ponto A-B
Sentimento de ação que você quer deixar (ex: "acreditar mais em si")
Define quais histórias entram e quais saem do roteiro

### 2. Navegação Hipnótica
Controle de plateia via entonação, velocidade, volume, pausas

### 3. Conteúdo Técnico + História
- Palestrante técnico: 90% conteúdo + 10% história
- Mas pessoas lembram dos 10%

### 4. Roteiro vs Texto
- Roteiro permite improvisação controlada
- Texto decora = artificial
- Roteiro + verdade = autenticidade

### 5. Compressão Temporal
Mesmo discurso em 30s, 5min, 20min, 1h
Ajusta densidade, não estrutura
