## Principais Aprendizados Preservados

### Metodologias/Frameworks

1. **Três Pilares da Palestra Poderosa**
   - Performance
   - Roteiro
   - Conexão Genuína

2. **Duas Maneiras de Falar**
   - SEM estrutura (fácil falar, difícil ser escutado)
   - COM estrutura (difícil falar, fácil ser escutado)

3. **Framework "3-4 Passos"**
   - Simplifique qualquer conteúdo em 3-4 passos
   - Clareza imediata + parece alcançável

4. **Princípio 10% a Mais**
   - Não precisa saber TUDO para ensinar
   - Só 10% a mais que quem quer aprender

5. **Técnica: Sair do "Eu"**
   - Incluir outros personagens
   - Fazer a plateia imaginar

6. **Linguagem Hipnótica**
   - "Talvez" ao invés de "todos"
   - CTA sempre (levanta a mão + fala EU)
   - Balançar cabeça positivamente

7. **Posicionamento no Palco**
   - Centro = autoridade
   - Microfone: 3 dedos abaixo do queixo
   - Nunca de costas para plateia
