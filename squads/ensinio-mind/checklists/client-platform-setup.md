# Checklist: Setup da Plataforma (Cliente Novo)

> Passo a passo para o cliente configurar sua plataforma Ensinio do zero.
> Baseado no vídeo oficial "Plataforma e App para Infoprodutores" — Gui Ávila (2026-03-14)
> Use para: CS guiar cliente, SDR mostrar que é simples, ou cliente seguir sozinho.

---

## Fase 1: Conta e Orientação (~5 min)

- [ ] Criar conta gratuita no plano Starter (ensinio.com)
- [ ] Fazer login e acessar o Admin Console (avatar → Admin Console)
- [ ] Entender os 3 ambientes: **Billboard** (/browse), **Admin Console** (gestão) e **Dentro do Grupo** (conteúdo)

**Dica:** A Billboard começa vazia. É normal. Você vai preenchê-la nos próximos passos.

---

## Fase 2: Criar Grupos (Seus Produtos) (~15 min)

- [ ] Admin Console → Grupos → Novo Grupo
- [ ] Definir título do grupo (nome do curso/produto)
- [ ] Definir forma de ingresso: **Pago** (padrão para produtos), Aberto ou Restrito
- [ ] Definir subtítulo (aparece na Billboard e página de vendas)
- [ ] Definir descrição (aparece na página de vendas gerada pela Ensinio)

**Repetir para cada produto.** Lembrar: grupo pode ser curso, mentoria, comunidade, e-book, assinatura.

---

## Fase 3: Identidade Visual dos Grupos (~30 min por grupo)

> Essa é a parte que mais demanda tempo, mas faz uma vez só.

### Artes obrigatórias (mínimo):
- [ ] **Capa vertical** (9:12) — para sessões com capas verticais na Billboard
- [ ] **Capa horizontal** (16:9) — para sessões horizontais na Billboard

### Artes recomendadas:
- [ ] **Estampa** (600x240) — logomarca/identidade do grupo. **Sem estampa = puxa título em texto (fica feio)**
- [ ] **Capa imersiva** — background da aba de aulas. **Usar versão da capa SEM textos**
- [ ] **Capa da comunidade** — banner do topo da comunidade do grupo
- [ ] **Imagens da Billboard** — imagem de fundo quando o grupo aparece como destaque

### Opcional:
- [ ] Versões mobile de cada arte (se não enviar, puxa a de desktop)

**Dica do Gui:** Pedir ao designer capa COM textos e SEM textos. A versão sem textos serve como capa imersiva e fundo de Billboard.

**Recurso:** Link para templates Canva com tamanhos corretos (descrição do vídeo).

---

## Fase 4: Conteúdo — Módulos e Aulas (~tempo varia)

- [ ] Entrar no grupo → aba Conteúdos
- [ ] Criar módulos (ex: Módulo 1, Módulo 2, Módulo 3)
- [ ] Dentro de cada módulo, criar aulas com:
  - [ ] Título da aula
  - [ ] Vídeo (upload via Ensinio Stream, YouTube ou live gravada)
  - [ ] Material complementar (PDF, áudio) — opcional
  - [ ] Quiz — opcional
- [ ] Escolher layout de aulas: **Imersivo** (padrão, poucos conteúdos), **Seriado** (muitas aulas) ou **Sequencial**

### Se usar Drip Content:
- [ ] Configurar liberação programada (imediata, após X dias, ou data específica)
- [ ] Estratégia recomendada: liberar bônus após período de garantia

---

## Fase 5: Billboard — Vitrine da Plataforma (~20 min)

### Destaques (parte nobre do topo):
- [ ] Admin Console → tela inicial → Destaques Iniciais
- [ ] Adicionar pelo menos 1 grupo como destaque
- [ ] Configurar vídeo de fundo no grupo (Configurações → Página Principal → Vídeo de destaque)
- [ ] Configurar kicker (texto curto acima do subtítulo, ex: "Curso + Comunidade")

### Sessões:
- [ ] Criar pelo menos 1 sessão (botão verde no Admin Console)
- [ ] Escolher tipo: Grupos (manual), Grupos em alta (automático), Módulos, Aulas mais assistidas
- [ ] Escolher orientação das capas: vertical ou horizontal
- [ ] Reordenar sessões conforme prioridade

### Trilhas (planos pagos):
- [ ] Criar trilha: Admin Console → Grupos e Recursos → Trilha
- [ ] Adicionar grupos na sequência desejada
- [ ] Adicionar trilha como sessão na Billboard

**Lembrar:** Alterações na Billboard demoram ~1 minuto para refletir (cache de performance).

---

## Fase 6: Comunidade (~10 min)

- [ ] Decidir se quer comunidade no grupo (vem ativa por padrão, pode desativar)
- [ ] Configurar página inicial do grupo: Comunidade, Aulas, Primeira aula ou Última assistida
- [ ] Configurar Feed central (Admin Console → Configurações → Comunidade):
  - [ ] Feed ativo? (recomendado: sim)
  - [ ] Permissão: todos os usuários ou membros de grupos específicos?
  - [ ] Criar sessões no feed (produtos em destaque)

---

## Fase 7: Vendas — Configurar Checkout (~15 min)

- [ ] Entrar no grupo → aba Vendas → Criar Oferta
- [ ] Definir tipo: pagamento único ou recorrência (mensal/semestral/anual)
- [ ] Definir preço
- [ ] Copiar link de checkout (white label: seudominio.com/payment/ID)
- [ ] Personalizar checkout: layout (one-page ou steps), cor do botão, banners, página de obrigado

### Vendas avançadas:
- [ ] Configurar Upsell (oferta pós-compra)
- [ ] Configurar Order Bump (oferta complementar no checkout)
- [ ] Configurar Vínculos entre grupos (pacotes de produtos)
- [ ] Ativar Parcelamento Inteligente (Admin Console → Vendas — recomendado manter ativo)
- [ ] Criar Cupons Inteligentes conforme estratégia de vendas

---

## Fase 8: Gamificação (~15 min)

- [ ] Admin Console → Grupos e Recursos → Gamificação → Ativar
- [ ] Configurar gatilhos (login, comentário, quiz, aula assistida) com pontos e limites diários
- [ ] Criar níveis (ex: Iniciante 2pts → Bronze 50 → Prata 100 → Ouro 200 → Diamante 1000)
- [ ] Upload de selos visuais por nível
- [ ] Ativar Ranking Geral (aparece na Billboard)
- [ ] (Opcional) Ativar Loja de Recompensas e cadastrar prêmios

---

## Fase 9: Aparência Global (~10 min)

- [ ] Admin Console → Configurações → Aparência e Design
- [ ] Upload logomarca (modo claro + modo escuro)
- [ ] Upload isotipo (símbolo da marca → vira favicon)
- [ ] Escolher modo de cor padrão: escuro (recomendado) com claro opcional
- [ ] Configurar domínio customizado (seudominio.com)
- [ ] Criar itens de menu customizado (links internos/externos no header)

---

## Fase 10: Agentes de IA (~10 min)

- [ ] Admin Console → Ensinio AI
- [ ] Criar agente (nome, tom de voz, diretrizes)
- [ ] Ativar habilidades: Vendas (liga/desliga), Reunião (liga/desliga)
- [ ] Conectar WhatsApp via QR code
- [ ] Conectar Google Agenda (para marcação de reuniões)
- [ ] Configurar base de conhecimento
- [ ] Testar conversando com o agente

---

## Validação Final

- [ ] Abrir plataforma em aba anônima (visão do aluno)
- [ ] Billboard com destaques, sessões e vídeo de fundo
- [ ] Entrar em um grupo: aulas carregam, layout bonito
- [ ] Checkout funciona (testar com compra de R$ 1)
- [ ] Comunidade ativa e feed configurado
- [ ] Gamificação dando pontos ao interagir
- [ ] Agente de IA respondendo no WhatsApp
- [ ] Testar no celular (artes mobile ok, menu colapsa)

---

## Tempo Total Estimado

| Tipo de cliente | Tempo estimado |
|----------------|---------------|
| **Mínimo viável** (1 grupo, sem gamificação) | ~2 horas |
| **Setup completo** (3+ grupos, gamificação, agentes) | ~4-6 horas |
| **Migração de outra plataforma** | +2-3 dias (com suporte Ensinio) |
