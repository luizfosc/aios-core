# Changelog — Luiz Fosc Mind Clone Squad

Todas as mudanças notáveis neste squad serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este squad adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2026-03-07
### Adicionado
- Integração total com o ecossistema **MBTI Premium**.
- 16 perfis de tipos de personalidade detalhados em `data/mbti/`.
- 15 novas tasks focadas em análise de personalidade, liderança e carreira.
- Heurísticas de MBTI (H008-H015) integradas ao Thinking DNA.
- Sistema de "Compatibility Matrix" para dinâmicas de equipe.

### Alterado
- Upgrade do **Thinking DNA** (v2.0 para v3.0) com foco em polimatia e neurodivergência.
- Expansão da base de dados (DNA) para ~203K palavras processadas.
- Melhoria nos prompts de "Voice DNA" para maior fidelidade de tom de voz.

### Corrigido
- Ajuste de densidade em tasks de MBTI para conformidade com o Quality Gate do Squad Creator Pro.

## [2.0.0] - 2025-11-15
### Adicionado
- Framework **Cinema-Mágica** para análise de palestras.
- Agente `luiz-fosc` com suporte a T0 (Diagnóstico).
- Primeira versão do **Methodology DNA** (Palestra de Elite).

## [1.0.0] - 2025-05-20
### Adicionado
- Lançamento inicial do Mind Clone Luiz Fosc.
- Base de dados básica com transcrições de vídeos e palestras.
- Tasks core de mentoria e revisão de conteúdo.
