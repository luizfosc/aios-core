# Tathi Deandhela Method Squad

Squad para diagnóstico, estruturação e evolução de palestras com base nos princípios da Tathi Deandhela.

## Status
- Versão: `1.0.0` (bootstrap)
- Modo: `quality-first`
- Fonte principal: corpus local "Palestrante Memorável"

## Estrutura
- `agents/`: orquestrador e especialistas.
- `tasks/`: execução prática por caso de uso.
- `workflows/`: fluxos com checkpoints e veto conditions.
- `data/`: índices de fontes e apoio de rastreabilidade.
- `docs/`: documentação operacional do squad.
