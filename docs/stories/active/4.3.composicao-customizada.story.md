# Story 4.3: Composição Customizada

## Description

As a **operador da skill politica-app**,
I want **instruções claras no SKILL.md para criar um preset demográfico customizado do zero**,
So that **eu possa simular cenários com composições demográficas específicas que não existem nos presets predefinidos**.

## Acceptance Criteria

- [x] AC1: SKILL.md contém seção dedicada "Como criar um preset customizado" com instruções passo a passo
- [x] AC2: Template `presets/custom-template.yaml` é guia funcional com todos os campos obrigatórios, valores de exemplo e comentários explicativos em cada campo
- [x] AC3: Instruções incluem: como definir segmentos, como escolher `template_overrides`, como validar que `count` soma o `total_agents`
- [x] AC4: Pelo menos 3 exemplos de cenários customizados documentados (ex: "só jovens 18-25", "região Norte", "classe empresarial")
- [x] AC5: Validação no pipeline: se preset customizado for fornecido, verificar schema e somar counts antes de prosseguir

## Dev Notes

- **Referência:** Spec Story 4.3 — "Instruções no SKILL.md para criar preset demográfico customizado sem usar os predefinidos"
- **Spec FR3:** "Permitir selecionar presets demográficos ou composição customizada"
- **O custom-template.yaml já existe como placeholder** (criado na Story 1.3). Aqui o foco é torná-lo um guia auto-explicativo
- **Foco em documentação:** Esta story é majoritariamente documentação e guia. O mecanismo de leitura de presets já existe nas stories anteriores
- **Cenários-exemplo sugeridos:**
  - "Capitais do Nordeste" (foco Recife, Salvador, Fortaleza)
  - "Jovens primeiro voto" (18-21 anos, digitais)
  - "Empresários e classe alta" (decisores econômicos)
