```
  └─ Uses unzipper, regex, validation
```

**Target Implementation:**
```
squads/ensinio-whatsapp-prospector/tasks/parse-chat-export.md
  └─ Imports from @ensinio/whatsapp-parser
  └─ Calls parseWhatsAppExport(zipPath)
  └─ Passes result to downstream tasks
```

### Module Usage Example

```javascript
const { parseWhatsAppExport, validateParsedData } = require('@ensinio/whatsapp-parser');

// In parse-chat-export.md execution:
const result = await parseWhatsAppExport(zipPath);
const validation = validateParsedData(result);

if (!validation.isValid) {
  throw new Error(`Invalid parse result: ${validation.errors.join(', ')}`);
}

// result.contacts, result.groupName, etc.
return result;
```
