# Query Original

## Pergunta do Usuário

> "Preciso que você faça uma pesquisa de quais os melhores livros de storytelling no Brasil, para eu colocar no meu material de palestras"

## Contexto Inferido

| Campo | Valor |
|-------|-------|
| **Domain** | products (recomendação de livros) |
| **Focus** | recommendation + curadoria para palestras |
| **Location** | Brasil (livros em português ou traduzidos) |
| **Temporal** | current (disponíveis agora) |
| **Use Case** | Material de referência para palestras |
