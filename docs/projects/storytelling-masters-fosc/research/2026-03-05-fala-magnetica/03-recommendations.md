# Recomendações

## Situação Atual

O termo **"Fala Magnética"** está **ocupado** no mercado brasileiro por múltiplos profissionais, com destaque para **Maurici de Oliveira** (Oficina da Fala Magnética — site próprio, LinkedIn ativo, 13 anos de experiência comprovada).

## Opções Estratégicas

### Opção 1: Usar o termo com diferenciação forte
- **Risco:** Médio — sem registro de marca confirmado, mas uso comercial existente
- **Requisito:** Diferenciação clara no posicionamento, público-alvo e metodologia
- **Ação:** Consultar advogado de propriedade intelectual para viabilidade de registro

### Opção 2: Criar variação única do termo
Exemplos de nomes não encontrados na pesquisa:
- "Fala que Magnetiza"
- "Voz Magnética"
- "Discurso Magnético"
- "Oratória de Impacto Magnético"
- **Vantagem:** Menor risco de conflito, mais fácil de registrar

### Opção 3: Criar nome completamente original
- **Vantagem:** Proteção total de marca, diferenciação absoluta
- **Desvantagem:** Perder o apelo do termo "magnético/a"

## Próximos Passos Recomendados

1. **Busca formal no INPI** — Verificar se "Fala Magnética" está registrada como marca (portal busca.inpi.gov.br)
2. **Análise de concorrência** — Avaliar qualidade/profundidade dos métodos existentes para identificar gaps
3. **Decisão de naming** — Escolher entre usar o termo (com risco) ou criar variação
4. **Consulta jurídica** — Se optar por usar o termo, consultar advogado de PI antes de lançar
