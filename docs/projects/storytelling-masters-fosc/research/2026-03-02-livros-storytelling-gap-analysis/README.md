# Gap Analysis — Livros de Storytelling para Palestras

**Domain:** `products` | **Data:** 2026-03-02 | **Status:** Completo

## TL;DR

Pesquisa de gap analysis sobre o catálogo existente de 37 livros de storytelling. Identificados **30 livros adicionais** que não estavam na lista, organizados em Top 15 (prioridade máxima) e Tier Secundário (15 adicionais). Principais lacunas: retórica clássica (0 livros), storytelling oral (apenas 1), oratória brasileira (apenas 1), e persuasão moderna (Cialdini de 1984 como único).
