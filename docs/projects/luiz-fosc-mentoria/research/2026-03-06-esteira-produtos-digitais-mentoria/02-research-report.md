## 2. Modelos de Entrega de Mentoria (2025-2026)

### Mapa Completo de Modelos

| Modelo | Formato | Preço Típico | Escalabilidade | Quem Referência |
|--------|---------|-------------|----------------|-----------------|
| **1:1 Coaching** | Calls privadas + Slack | $49-$1.000/hora | Baixa (bandwidth humano) | GrowthMentor |
| **Group Coaching** | Calls em grupo + Q&A + conteúdo | $97-$497/mês | Média | Amy Porterfield |
| **Mastermind** | Grupos pequenos (7-10) + peer learning | $2.497-$50.000/ano | Média-Baixa | Brunson, Polish |
| **Bootcamp/Intensivo** | Trimestral, 1-3 dias, alta intensidade | $5.995+/sessão | Média | Dan Sullivan |
| **Membership/Assinatura** | Comunidade + conteúdo recorrente | $29-$499/mês | Alta | Stu McLaren |
| **Cohort-Based Course** | Turmas fechadas + prazos + comunidade | $500-$5.000/turma | Alta | Wes Kao (Maven) |
| **Híbrido Premium** | Coaching + grupo + comunidade | $2.497/mês+ | Média-Alta | The Collective |
| **Ultra-Exclusivo** | Eventos presenciais + acesso direto | $25K-$100K/ano | Muito Baixa | Genius Network, Inner Circle |

### Tendências 2025-2026

1. **De 3 meses para 12 meses** — Programas mais longos com precificação mais alta e resultados melhores
2. **Eventos presenciais como funil** — O método #1 para preencher programas high-ticket (>$3K) é evento ao vivo
3. **AI-augmented coaching** — Modelos híbridos combinando mentores humanos com IA para escala e consistência
4. **Métricas de transformação** — Coaches de elite medem progresso, engajamento e sentimento emocional, não só receita
5. **Mastermind ≠ Group Coaching** — Distinção cada vez mais clara: masterminds são problem-solving, group coaching é content-based

## 3. Estratégias de Precificação

### Value Ladder Clássico (Brunson/Hormozi)

```
FREE          → Lead Magnet (ebook, webinar, checklist)
$27-$97       → Tripwire / Low-Ticket (mini-curso, template)
$297-$997     → Core Offer (curso completo, programa)
$2.000-$10K   → High-Ticket (coaching em grupo, mastermind)
$10K-$50K+    → Ultra-Premium (1:1, inner circle, retainer)
```

**Conversão esperada:** Cliente de $27 pode virar cliente de $2.000 (235x lifetime value increase)

### Value Grid — Evolução 2025 (Hormozi)

O **Value Grid** é a evolução do Value Ladder. Em vez de assumir que o cliente sobe linearmente, oferece **múltiplas opções no mesmo nível de preço** para necessidades diferentes:

```
Nível de Preço    │ Opção A          │ Opção B          │ Opção C
──────────────────┼──────────────────┼──────────────────┼──────────────
$97/mês           │ Membership       │ Templates Pack   │ Community
$997              │ Curso Completo   │ Done-For-You     │ Bootcamp
$5.000+           │ Mastermind       │ 1:1 Coaching     │ VIP Day
```

### Psicologia de Precificação (3 Tiers)

- **Tier baixo:** Ancora o preço mínimo
- **Tier médio:** Onde a maioria compra (efeito âncora racional)
- **Tier alto:** Ancora o valor premium, valida os tiers inferiores
- **Payment plans** aumentam conversão sem reduzir preço (ex: 3x de $997 vs $2.497 à vista)

## 5. Críticas e Limitações (Devil's Advocate)

### Dados de Alerta

| Métrica | Valor | Fonte |
|---------|-------|-------|
| Taxa de sucesso de cursos online | ~10% | Indústria |
| Programa high-ticket sem refund | $30.000 (reportado) | Mayra Leen |
| Empresas com coaching efetivo | 42% (de 86% que consideram importante) | ICF 2020 |
| Empresas com succession planning forte | 14% | Deloitte |

### Problemas Estruturais

1. **1:1 não escala** — Bandwidth humano é o gargalo fundamental
2. **10% de sucesso** — A maioria dos programas de coaching falha em entregar resultados significativos
3. **Manipulação emocional** — Programas high-ticket usam táticas de dependência em vez de empoderamento
4. **Vetting inexistente** — "Aceitam TODOS que têm cartão de crédito" (Mayra Leen)
5. **Modelo de ontem na realidade de hoje** — Muitos coaches rodam modelos obsoletos
6. **Promessas exageradas** — "Rápido e simples" esconde custos ocultos reais
7. **Dependency creation** — Esteiras bem feitas empoderam autonomia; mal feitas criam dependência perpétua

### Antídotos (O Que Funciona)

- "Desacelerar antes de acelerar" — Validar antes de escalar
- Vetting rigoroso de clientes (não aceitar todos)
- Métricas de transformação, não só revenue
- Programas de 12 meses com milestones claros
- Modelos híbridos AI + humano para escala sem perder qualidade
