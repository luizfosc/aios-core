# Consulta de Marca no INPI

**Domain:** general | **Date:** 2026-03-05 | **Confidence:** HIGH

## TL;DR

Sim, e possivel consultar gratuitamente se uma marca tem registro no INPI atraves da base de dados oficial em https://busca.inpi.gov.br/pePI/. Nao requer login nem pagamento. O registro custa R$ 440-880 por classe e leva 12-24 meses.
