# Recomendacoes

## Para Consulta Rapida (Gratuita)

1. Acesse https://busca.inpi.gov.br/pePI/
2. Use **busca radical** com o nome da marca
3. Verifique a **classe NCL** do seu segmento
4. Se encontrar marca com status "em vigor" na mesma classe, ha conflito potencial

## Para Registro

1. Faca a busca gratuita primeiro para verificar viabilidade
2. Se viavel, acesse https://gru.inpi.gov.br/emarcas/ (requer login GOV.BR)
3. Pague a GRU (R$ 440 com desconto MEI/ME/EPP, R$ 880 sem desconto)
4. Acompanhe o processo pelo numero de 9 digitos

## Quando Contratar Profissional

- Se a busca retornar marcas similares (mas nao identicas) na mesma classe
- Se precisar de analise de similaridade fonetica/visual/conceitual
- Se o registro for estrategico para o negocio (custo: R$ 1.500-5.000)
